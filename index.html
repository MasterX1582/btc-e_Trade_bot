Btc-e trade bot

A python trading bot for btc-e

Download as .zip Download as .tar.gz View on GitHub

BETA 0.1.6

Uses alanmcintyre's btc-e api to automatically buy/sell a chosen currency from btc-e. found at: https://github.com/alanmcintyre/btce-api

Usage

To use the bot, just put your api key and secret into the config file, then run auto.py with python 2.7 Also you may edit other variables such as trade threshold The testing folder is for testing releases that may or may not be working, try them at your own risk
